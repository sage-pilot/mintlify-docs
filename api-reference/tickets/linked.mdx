Get ticket chats linked by `tracker_id`.

## Required access

| Authentication | Scope | Dataset grant |
|----------------|-------|---------------|
| [API client v2](/api-reference/authentication/api-client-v2) | Any ticket scope: `tickets:read`, `tickets:write`, `tickets:assign`, or `tickets:reply` | `tickets` |

## Query parameters

<ParamField query="tracker_id" type="string" required>
  Tracker ID used to find linked ticket chats.
</ParamField>

## Response

<ResponseField name="[]" type="object[]" required>
  Linked ticket chat objects.
</ResponseField>
