Create a customer, or return the existing customer if the email or phone already exists.

## Required access

| Authentication | Scope | Dataset grant |
|----------------|-------|---------------|
| API client v2 | `customers:write` | `customers` |

<Warning>
API client v2 key generation currently exposes `customers:read` for customer APIs. Use this endpoint only when your API client has been separately provisioned with `customers:write`.
</Warning>

## Body

<ParamField body="channel_id" type="string" required>
  Channel ID to associate with the customer.
</ParamField>

<ParamField body="name" type="string">
  Customer name.
</ParamField>

<ParamField body="email" type="string">
  Customer email. Provide at least `email` or `phone`.
</ParamField>

<ParamField body="phone" type="string">
  Customer phone number. Provide at least `email` or `phone`.
</ParamField>

## Response

<ResponseField name="customer" type="object" required>
  Created or existing customer object.
</ResponseField>

<ResponseField name="created" type="boolean" required>
  `true` when a new customer was created; `false` when an existing customer was returned.
</ResponseField>
