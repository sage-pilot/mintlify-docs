Get recent chats for a customer in the authenticated workspace.

## Required access

| Authentication | Scope | Dataset grant |
|----------------|-------|---------------|
| [API client v2](/api-reference/authentication/api-client-v2) | `customers:read` | `customers` |

## Body

<ParamField body="customerId" type="string" required>
  Customer ID to fetch chats for.
</ParamField>

<ParamField body="workspaceId" type="string">
  Optional workspace ID. The authenticated workspace is used by default.
</ParamField>

<ParamField body="includeLinkedContacts" type="boolean" default="false">
  Include chats from linked contacts.
</ParamField>

## Query parameters

<ParamField query="page" type="integer" default="1">
  Page number.
</ParamField>

<ParamField query="page_size" type="integer" default="20">
  Records per page. Maximum value is `200`.
</ParamField>

## Response

<ResponseField name="[]" type="object[]" required>
  Recent chat records for the customer.
</ResponseField>
