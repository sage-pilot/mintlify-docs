Use this endpoint template for the analytics query API. Full request and response details will be added after the final contract is confirmed.

## Required access

| Authentication | Scope | Dataset grant |
|----------------|-------|---------------|
| [API client v2](/api-reference/authentication/api-client-v2) | `analytics.query` | The dataset used in the request body, such as `tickets`, `messages`, `outbound_messages`, `marketing_attribution`, `csat`, or `workforce_activity`. |

## Body

<ParamField body="dataset" type="string" required>
  Dataset to query.
</ParamField>

<ParamField body="metrics" type="string[]" required>
  Metrics to calculate.
</ParamField>

<ParamField body="dimensions" type="string[]">
  Optional dimensions to group results.
</ParamField>

<ParamField body="filters" type="object">
  Optional filter object.
</ParamField>

<ParamField body="date_range" type="string">
  Optional date preset.
</ParamField>

<ParamField body="timezone" type="string">
  Timezone used to interpret date ranges and formatted datetime fields.
</ParamField>

## Notes

- This endpoint uses API client v2 credentials. See [API client v2](/api-reference/authentication/api-client-v2).
