<Panel>

<RequestExample>

```bash cURL
curl -X POST "https://app.sagepilot.ai/send-whatsapp-template" \
  -H "Authorization: Bearer <legacy_workspace_api_key>" \
  -H "X-SP-Workspace-Id: <workspace_id>" \
  -H "Content-Type: application/json" \
  -d '{
    "channel_id": "<channel_id>",
    "customer_phone": "<customer_phone>",
    "template": {
      "name": "<template_name>",
      "language": "en"
    }
  }'
```
