The Instagram channel connects Instagram Business messaging and comment workflows to Sagepilot. Use it when your support or social team needs one place to manage Instagram DMs, comment replies, and AI-assisted responses.

## Before you start

- Confirm the Instagram account is a Business account connected through Meta.
- Make sure you have admin access to the connected Meta assets.
- Select a default AI agent for DMs.
- Select a social agent if comments should be reviewed or answered by AI.
- Decide whether Sagepilot should reply to, hide, or delete comments.

## Create the Instagram channel

<Steps>
  <Step title="Open Channels">
    In Sagepilot, open **Channels** and click **Create Channel**.
  </Step>
  <Step title="Select Instagram">
    Choose **Instagram** from the Social category.
  </Step>
  <Step title="Set channel ownership">
    Add the channel name, select the default AI agent, choose human assignment behavior, and choose the default team.
  </Step>
  <Step title="Connect Instagram">
    Click **Connect Instagram Account** and complete the Meta authorization flow.
  </Step>
  <Step title="Configure comments">
    Choose the social agent and comment actions Sagepilot is allowed to perform, such as reply, hide, or delete.
  </Step>
  <Step title="Configure follow-ups and auto-close">
    Review the follow-up and auto-close settings so stale social conversations do not stay open forever.
  </Step>
  <Step title="Test the channel">
    Send a DM and add a test comment from a non-admin account.
  </Step>
</Steps>

## Recommended configuration

| Area | Recommendation |
| --- | --- |
| Default AI agent | Use the support agent that should handle private DMs. |
| Social agent | Use a social agent for public comment moderation and replies. |
| Human assignment | Enable when public or sensitive issues should escalate to the social/support team. |
| Comment actions | Start with reply-only, then enable hide/delete after internal review. |

## Verify the setup

1. Send an Instagram DM to the connected account.
2. Confirm the DM appears in Sagepilot.
3. Add a test comment on a post.
4. Confirm comment rules behave as expected.
5. Escalate a social conversation and confirm it routes to the right team.

## Operational notes

- Meta permissions can change if the authorizing user loses admin access.
- Public comment automation should be reviewed carefully before launch.
- Keep social-agent instructions separate from private DM support instructions.
