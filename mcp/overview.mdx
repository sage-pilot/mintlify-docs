Sagepilot MCP lets AI clients query workspace analytics, search support data, inspect customers and tickets, list channels and agents, and perform supported support actions.

## Server URL

```text
https://app.sagepilot.ai/platform-api/mcp
```

EU-hosted workspaces:

```text
https://eu.sagepilot.ai/platform-api/mcp
```

## Requirements

Your MCP client must support:

1. Remote Streamable HTTP.
2. Custom HTTP headers.
3. `Authorization: Bearer ...`.

If a client says `auth unsupported`, it usually means that client UI expects interactive OAuth connector setup. Sagepilot MCP uses API client tokens instead. Use a header-token MCP configuration.
